# R-Tree for Fortran

Sort-Tile-Recursive (STR) bidimensional R-Tree implementation written in modern Fortran.

## Features
- STR packing (Leutenegger et al. 1997)

## Compilation

### Build Requirements
- Compiler: `gfortran` (Standard Fortran 2008+)
- Build Tool: `make`

### Standard Build
`make`

### Debug Build
`make DEBUG=1`

### Testing
`make test`
